# 微信群聊消息抓取工具

## 项目说明

本工具用于抓取微信桌面版的群聊消息，通过OCR识别屏幕内容，解析出发送人、时间、目的地、物品信息，并导出到Excel表格。

## 技术方案

- **屏幕框选**: 鼠标拖拽框选微信聊天区域，确保只捕获目标窗口
- **OCR识别**: 使用Tesseract识别中文文字
- **消息解析**: 基于关键词和正则表达式提取关键信息
- **Excel导出**: 使用pandas+openpyxl生成Excel表格
- **配置保存**: 区域设置自动保存，下次启动自动加载

## 安装依赖

```bash
pip install -r requirements.txt
```

还需要安装Tesseract OCR引擎：

1. 下载Tesseract: https://github.com/UB-Mannheim/tesseract/wiki
2. 安装后确保路径正确（代码中已配置默认路径）

## 使用方法

### 首次使用

1. 双击 `启动抓取工具.bat` 启动程序
2. 点击 **📐 框选抓取区域** 按钮
3. 屏幕会变黑，用鼠标拖拽框选微信的聊天区域
4. 框选完成后自动返回，区域已保存
5. 点击 **📸 立即抓取** 即可

### 日常使用

- 双击 `启动抓取工具.bat` 启动程序
- 程序会自动加载上次的区域配置
- 点击 **📸 立即抓取** 或 **🔄 自动监控**

## 功能说明

| 按钮 | 功能 |
|------|------|
| 📐 框选抓取区域 | 鼠标框选微信聊天区域 |
| 🔄 刷新区域 | 手动输入坐标后保存 |
| 📸 立即抓取 | 执行一次抓取 |
| 🔄 自动监控 | 每5秒自动抓取一次 |
| ⏹ 停止 | 停止自动监控 |
| 📁 查看结果 | 打开Excel输出目录 |

## 配置说明

配置文件 `capture_config.json` 会自动保存在程序目录，包含：
- 抓取区域的X、Y坐标和宽高
- 窗口标题（默认"微信"）

## 关键词配置

### 目的地关键词
到、发往、去、送、前往、至、去往、送往、发到

### 物品关键词
货、物品、产品、货品、件、箱、条、个、台、套、批、吨

### 物流关键词
车牌、司机、运费、吨位、重量、车型、装货、卸货、地址、目的地

## 注意事项

1. 微信窗口需要保持在桌面可见状态
2. 首次使用必须框选抓取区域
3. OCR识别效果受屏幕分辨率和文字清晰度影响
4. 建议微信字体设置为标准大小以提高识别率
5. 自动监控模式下，微信窗口不能最小化
