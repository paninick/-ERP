                微信群消息抓取工具 v4.0
═══════════════════════════════════════════════════════════════

【版本说明】
基于 wxauto 方案，识别精准，支持自动发送微信

【环境要求】
- Windows 10/11 或 Windows Server
- Python 3.9+
- 微信PC版 3.9.x

【快速开始】

1. 双击 "启动.bat" 运行程序
   （会自动安装 wxauto 等依赖）

2. 首次运行会弹出微信授权，点击"允许"

3. 点击 "刷新群列表" 选择要监控的群

4. 配置发送目标（可选）：
   - 输入收件人微信号/昵称
   - 勾选"生成Excel后自动发送"

5. 点击 "立即抓取" 或 "自动监控"

【功能说明】

┌────────────────┬─────────────────────────────────────┐
│ 刷新群列表     │ 获取微信群聊列表                   │
│ 立即抓取       │ 执行一次抓取                       │
│ 自动监控       │ 持续自动抓取（间隔可调）           │
│ 启用去重       │ 过滤重复消息                       │
│ 生成Excel后    │ 勾选后，完成抓取自动发送Excel     │
│   自动发送     │ 给指定人员                         │
└────────────────┴─────────────────────────────────────┘

【输出文件】
位置：程序目录下的 output 文件夹
格式：微信记录_YYYYMMDD.xlsx
字段：时间、发送人、目的地、物品、原始消息

【打包EXE】
双击 "build_exe.bat" 打包成单文件
生成位置: dist\微信群消息抓取.exe

【常见问题】

Q: 提示"wxauto未安装"
A: 运行: pip install wxauto

Q: 获取不到群列表
A: 确保微信已打开并登录

Q: 自动发送失败
A: 检查"发给"是否填写正确（填微信号或昵称）
