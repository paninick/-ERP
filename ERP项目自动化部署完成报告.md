## 七、注意事项

### 7.1 保持服务运行

- MySQL 服务需要保持运行
- Redis 服务需要保持运行
- 若依应用需要保持运行

### 7.2 数据备份

建议定期备份数据库：
```bash
mysqldump -u root -ppassword ry-vue > backup.sql
```

### 7.3 日志路径

由于日志配置使用了 Linux 路径 `/home/ruoyi/logs/`，在 Windows 上可能有警告，但不影响系统运行。

如需修改日志路径，编辑：
`d:\erp\RuoYi-Vue\ruoyi-admin\src\main\resources\logback.xml`
