# ERP 项目 - 轻量级部署方案（H2 嵌入式数据库）

**日期**：2026-04-02  
**方案类型**：轻量级部署，对笔记本性能占用最小

---

## 一、方案概述

### 1.1 轻量级方案优势

| 特性 | 说明 |
|------|------|
| **无需外部服务** | 不需要 MySQL、Redis 等外部服务 |
| **内存占用小** | H2 + 嵌入式缓存，总内存占用 &lt; 512MB |
| **启动快速** | 无需等待外部服务启动，直接启动应用 |
| **零配置** | 开箱即用，无需配置数据库 |
| **数据持久化** | 数据保存在本地文件中 |

### 1.2 技术栈

| 组件 | 说明 |
|------|------|
| **数据库** | H2 嵌入式数据库（文件存储） |
| **缓存** | Caffeine（Java 本地缓存，替代 Redis） |
| **应用** | Spring Boot 4.0.3 |

## 二、H2 数据库配置

### 2.1 添加 H2 依赖

在 `ruoyi-admin/pom.xml` 中添加：

```xml
<dependency>
    <groupId>com.h2database</groupId>
    <artifactId>h2</artifactId>
    <scope>runtime</scope>
</dependency>

<dependency>
    <groupId>com.github.ben-manes.caffeine</groupId>
    <artifactId>caffeine</artifactId>
</dependency>
```

### 2.2 创建 H2 数据库配置

创建文件 `ruoyi-admin/src/main/resources/application-h2.yml`：

```yaml
spring:
  datasource:
    type: com.alibaba.druid.pool.DruidDataSource
    driver-class-name: org.h2.Driver
    url: jdbc:h2:file:./data/ry-vue;DB_CLOSE_ON_EXIT=FALSE;AUTO_SERVER=TRUE
    username: sa
    password:
  
  h2:
    console:
      enabled: true
      path: /h2-console

  data:
    redis:
      host: localhost
      port: 6379
      database: 0
      lettuce:
        pool:
          max-active: 8
          max-wait: -1ms
          max-idle: 8
          min-idle: 0
```

### 2.3 使用 H2 配置启动

修改 `application.yml`，激活 H2 profile：

```yaml
spring:
  profiles:
    active: h2
```
