## 五、技术要点记录

### 5.1 验证注解使用规范

**金额字段（BigDecimal）**：
```java
@Digits(integer = 10, fraction = 4, message = "金额格式不正确")
@DecimalMin(value = "0", message = "金额不能为负数")
private BigDecimal amount;
```

**比率字段（百分比）**：
```java
@Digits(integer = 3, fraction = 4, message = "比率格式不正确")
@DecimalMin(value = "-100", message = "比率不能小于-100%")
@DecimalMax(value = "100", message = "比率不能大于100%")
private BigDecimal rate;
```

**数字字段（Integer）**：
```java
@NotNull(message = "数量不能为空")
@Min(value = 1, message = "数量必须大于0")
private Integer qty;
```

**字符串字段**：
```java
@NotBlank(message = "名称不能为空")
@Size(max = 200, message = "名称长度不能超过200个字符")
private String name;
```
