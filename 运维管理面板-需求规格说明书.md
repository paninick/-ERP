## 1. 项目概述

### 1.1 项目背景
针对笔记本非正常关闭可能导致的系统异常问题，开发一个功能完善、用户友好的运维管理面板，提升测试环境的稳定性和运维效率。

### 1.2 项目目标
- 使任何技术水平的人员都能快速识别系统异常
- 提供有效的措施恢复业务
- 提升测试环境的稳定性和运维效率

### 1.3 目标用户
- 维护工程师
- 项目经理
- 非专业运维人员（包括技术小白）
- 测试人员

## 7. 技术选型

### 7.1 后端技术栈
- **框架**：Spring Boot 2.7.x
- **语言**：Java 17
- **数据库**：MySQL 8.0
- **缓存**：Redis 6.x
- **WebSocket**：Spring WebSocket
- **系统监控**：OSHI（Java）/ PSUtil（Python）
- **服务健康检查**：Spring Boot Actuator

### 7.2 前端技术栈
- **框架**：Vue 2.x
- **UI组件库**：Element UI
- **图表库**：ECharts 5.x
- **状态管理**：Vuex
- **路由**：Vue Router
- **构建工具**：Webpack

### 7.3 部署架构
- **应用服务器**：Nginx + Tomcat
- **数据库**：MySQL主从复制
- **缓存**：Redis Sentinel
- **监控**：Prometheus + Grafana（可选）
