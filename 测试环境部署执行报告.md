## 四、后续部署步骤

### 4.1 下一步需要准备的环境

#### 1. 启动 MySQL 服务

```bash
# 检查 MySQL 是否安装
mysql --version

# 启动 MySQL 服务（Windows）
# 方式1：使用服务管理器
# 方式2：命令行
net start MySQL80
```

#### 2. 创建数据库并导入数据

```sql
-- 创建数据库
CREATE DATABASE IF NOT EXISTS `ry-vue` 
DEFAULT CHARACTER SET utf8mb4 
COLLATE utf8mb4_general_ci;

-- 导入基础数据
USE `ry-vue`;
SOURCE d:/erp/RuoYi-Vue/sql/ry_20260321.sql;
SOURCE d:/erp/RuoYi-Vue/sql/quartz.sql;
SOURCE d:/erp/RuoYi-Vue/ruoyi-demo/src/main/resources/sql/init.sql;
```

#### 3. 启动 Redis 服务

```bash
# 检查 Redis 是否安装
redis-cli --version

# 启动 Redis 服务（Windows）
# 方式1：使用 redis-server.exe
# 方式2：使用服务管理器
```

#### 4. 配置应用

修改配置文件（如需要）：

**数据库配置**（`ruoyi-admin/src/main/resources/application-druid.yml`）：
```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/ry-vue?useUnicode=true&characterEncoding=utf8&zeroDateTimeBehavior=convertToNull&useSSL=true&serverTimezone=GMT%2B8
    username: root
    password: your_password  # 修改为您的密码
```

**Redis 配置**（`ruoyi-admin/src/main/resources/application.yml`）：
```yaml
spring:
  data:
    redis:
      host: localhost
      port: 6379
      password:  # 如果有密码请填写
      database: 0
```

#### 5. 启动应用

```bash
# 方式1：使用 JAR 包启动
cd d:\erp\RuoYi-Vue\ruoyi-admin
java -jar target\ruoyi-admin.jar

# 方式2：使用 Maven 启动（开发环境）
cd d:\erp\RuoYi-Vue\ruoyi-admin
mvn spring-boot:run
```
