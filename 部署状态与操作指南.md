## 三、操作指南

### 3.1 启动 MySQL 服务

#### 方式1：使用 MySQL Installer（推荐）

如果您使用 MySQL Installer 安装的 MySQL：

1. 打开 **MySQL Installer**
2. 点击 **Reconfigure** 或 **Configure**
3. 找到 **MySQL Server** 组件
4. 点击 **Start** 或 **Configure**
5. 按照向导配置并启动服务

#### 方式2：使用 Windows 服务管理器

1. 按下 `Win + R`
2. 输入 `services.msc` 并回车
3. 查找名称中包含 "MySQL" 的服务
4. 如果找到，右键 → **启动**

#### 方式3：手动启动 MySQL Server

如果 MySQL 服务没有注册为 Windows 服务：

```bash
# 以管理员身份打开命令提示符
cd "C:\Program Files\MySQL\MySQL Server 8.4\bin"

# 初始化数据目录（如果是首次安装）
mysqld --initialize --console

# 安装为 Windows 服务
mysqld --install MySQL84

# 启动服务
net start MySQL84
```

### 3.2 验证 MySQL 服务启动

```bash
# 方式1：检查服务状态
sc query MySQL84

# 方式2：测试连接
& "C:\Program Files\MySQL\MySQL Server 8.4\bin\mysql.exe" -u root -ppassword -e "SELECT VERSION();"

# 应该显示 MySQL 版本：8.4.8
```

### 3.3 创建数据库并导入数据

MySQL 服务启动后，执行以下步骤：

```bash
# 1. 创建数据库
& "C:\Program Files\MySQL\MySQL Server 8.4\bin\mysql.exe" -u root -ppassword -e "CREATE DATABASE IF NOT EXISTS \`ry-vue\` DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;"

# 2. 导入基础数据
& "C:\Program Files\MySQL\MySQL Server 8.4\bin\mysql.exe" -u root -ppassword ry-vue < d:\erp\RuoYi-Vue\sql\ry_20260321.sql

# 3. 导入定时任务数据
& "C:\Program Files\MySQL\MySQL Server 8.4\bin\mysql.exe" -u root -ppassword ry-vue < d:\erp\RuoYi-Vue\sql\quartz.sql

# 4. 导入 Demo 模块数据
& "C:\Program Files\MySQL\MySQL Server 8.4\bin\mysql.exe" -u root -ppassword ry-vue < d:\erp\RuoYi-Vue\ruoyi-demo\src\main\resources\sql\init.sql

# 5. 验证数据
& "C:\Program Files\MySQL\MySQL Server 8.4\bin\mysql.exe" -u root -ppassword ry-vue -e "SHOW TABLES; SELECT COUNT(*) FROM sys_user; SELECT COUNT(*) FROM demo_order;"
```

### 3.4 启动应用

数据库和数据准备完成后，启动应用：

```bash
# 方式1：使用 JAR 包
cd d:\erp\RuoYi-Vue\ruoyi-admin
java -jar target\ruoyi-admin.jar

# 方式2：使用 Maven
cd d:\erp\RuoYi-Vue\ruoyi-admin
mvn spring-boot:run
```
