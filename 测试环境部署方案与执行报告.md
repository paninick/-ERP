### 2.2 应用配置

**application.yml 关键配置**：

```yaml
# 服务器端口
server:
  port: 8080

# 数据库配置（application-druid.yml）
spring:
  datasource:
    type: com.alibaba.druid.pool.DruidDataSource
    driver-class-name: com.mysql.cj.jdbc.Driver
    url: jdbc:mysql://localhost:3306/ry-vue?useUnicode=true&characterEncoding=utf8&zeroDateTimeBehavior=convertToNull&useSSL=true&serverTimezone=GMT%2B8
    username: root
    password:

# Redis 配置
  data:
    redis:
      host: localhost
      port: 6379
      database: 0

# 文件上传路径
ruoyi:
  profile: D:/ruoyi/uploadPath
```

## 六、常见问题处理

### 6.1 数据库连接问题

**问题**：无法连接到 MySQL 数据库

**解决方案**：
1. 检查 MySQL 服务是否启动
2. 检查数据库用户名和密码
3. 检查数据库 URL 配置
4. 检查防火墙设置

### 6.2 Redis 连接问题

**问题**：无法连接到 Redis

**解决方案**：
1. 检查 Redis 服务是否启动
2. 检查 Redis 地址和端口
3. 检查 Redis 密码配置
4. 检查防火墙设置

### 6.3 应用启动失败

**问题**：应用启动时报错

**解决方案**：
1. 检查日志文件，定位错误
2. 检查数据库连接配置
3. 检查 Redis 连接配置
4. 检查端口是否被占用
